Setup & Instructions: (for Windows OS)
1. Have Git installed as a path. 
2. Have Python 3 and PIP installed.
3. Install this: "pip install git+https://github.com/yuce/pyswip@master#egg=pyswip".
4. Download the zip file.
5. Unzip the folder.
6. Open a command line on the folder's directory.
7. Run the command: py interface.py
8. Type in valid prompts and wait for a reply.
9. To exit the program, type "exit".
